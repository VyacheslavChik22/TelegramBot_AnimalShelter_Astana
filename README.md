### Командный проект студентов SKY.pro Java dev 6 поток. Telegram-bot для приюта животных
Приложение Telegram-бот, который сможет отвечать на вопросы от людей о том, что нужно знать и уметь, чтобы взять животное из приюта, запрашивать ежедневный отчет от хозяев.
